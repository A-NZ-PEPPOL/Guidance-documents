# Welcome to the A-NZ Peppol Framework Guidance Hub

This hub provides a set of guides to support the development of Peppol Access Points and Service Metadata Publishers in alignment with Australian and New Zealand (A-NZ) compliance requirements.

### Available Guidance Documents:

- [GN01: Including WET and LCT on an Invoice](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote01_WETandLCT):
  Guidance on correctly representing Wine Equalisation Tax (WET) and Luxury Car Tax (LCT) within Peppol einvoices.
  
- [GN02: Payment means and use of the UNCL4461 code list](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote02_Payment_Means_Code.html): Guidance on applying Peppol payment means and using the UNCL4461 code list within A‑NZ einvoicing.
  
- [GN03: A-NZ Information Security Guidance for e-Invoicing Service Providers](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote03_Information_Security.html): Guidance outlining the information security requirements that A‑NZ Peppol service providers must meet for accreditation and continued compliance.
  
- [GN08: Peppol Participant, Party, Process and Document Identifiers](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote08_UseOfIdentifiers.html): Comprehensive guidance on identifier structure and correct use within the A‑NZ Peppol network.
  
- [GN09: Peppol Directory](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote09_PeppolDirectory.html): Guidance to ensure accurate and consistent A‑NZ data in the Peppol Directory.
  
- [GN10: End User Identification](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote10_EndUserIdentification.html): Guidance on how Service Providers in Australia and New Zealand can meet the requirements of End User Identification in the Peppol environment.
  
- [GN11: Use of Tax Category Codes in Australia](https://a-nz-peppol.github.io/Guidance-documents/GuidanceNote11_Use_of_Tax_Category_Codes.html): Guidance on using Peppol tax category codes for Australian tax invoices and credit notes.
